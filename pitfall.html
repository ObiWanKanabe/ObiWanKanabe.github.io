PitFall



At a glance

I created Pitfall during my 4th semester object-orientated analysis and design class at Humber College. Written in C++ and SDL2, I used the creation of this game to learn how to begin orgnaizing my code in my projects.

Features

Current main features in the most recent release include: player movement, sprite animation, one difficult map, different menus, and fully playable game.

My Contribution

During the creation of the Pitfall and the object-ortientated analysis and design class, I learned how to use UML diagrams to properly plan a project, and then follow through. The one map was made to be difficult to challenge my friends, and the video below will demonstrate a small portion of the game.

Whats Next

The features I would add to the game include: More levels, nicer textures for the map, and audio.
