Sequential




At a glance

I created Sequential during my 2nd year at Humber College. Created used Unreal Engine 4, I used the creation of this game to learn how to use Blueprints in a simple gameplay setting. The map was created by project partner Miguel Favero.

Features

Current main features in the project include: multiplayer, character movement, audio, character animation, and blueprints.

My Contribution

During the creation of the game, I learned how to use set up a multiplayer Unreal Engine project, as well as using networked blueprints to ensure the game is properly playable with two players. The short video below will demonstrate how the game plays and the first puzzle room being completed.
